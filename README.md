# Hyesi

This is the game I made for the Texas A&M Game Developers Spring 2026 Game Jam.
It won **3rd Place Overall** and **Best in Art**

The code here is a cleaned version of the actual repo I used, it removed any external assets like fonts & addons.


## About
A 2D platformer with a roguelite augment system, I took inspiration from Ori Will of Wisps for the art direction.
